Broken Motion

Project Type: Thesis Project

Game Type: Video Game

Software: Unity, Visual Studio, Blender

Coding Languages: C#

Broken Motion is a Roguelike FPS that explores player movement and ability synergy.

  • Physics-based Movement System
  • FPS Weapon System
  • Ability and Unlock Systems
  • ScriptableObjects (cards)
  • Enemy Spawning and Behaviors
  • Component of Gradute Thesis (will link when published)




Reflection

What is this project?

Broken Motion is a Roguelike FPS inspired by games like Titanfall 2, Roboquest and Slay the Spire.

This project was made entirely solo, including all visuals, art, programming and game design.
